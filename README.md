# Proyecto-Banca-PalParne

Este proyecto es una aplicación sencilla hecha en Python que simula un banco básico.  
Es un trabajo para las clases de Python y sirve para practicar lo que hemos visto en clase como funciones, módulos, condicionales y bucles.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

Con este programa puedes:

- Ver el saldo que tienes
- Ingresar dinero
- Retirar dinero
- Enviar dinero a otra cuenta
- Registrar gastos
- Ver los gastos guardados
- Calcular la media de los gastos

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

Archivos con los que esta escrito:

- main.py  
  Es el archivo principal, desde aquí se arranca el programa.

- MenuPrincipal.py  
  Tiene el menú y controla las opciones que elige el usuario.

- EnviarDinero.py  
  Aquí están las funciones para ingresar, retirar, enviar dinero y consultar el saldo.

- Usuario.py  
  Contiene la estructura básica de un usuario.

- registro_gastos1.py y registro_gastos2.py  
  Se usan para gestionar y guardar los gastos.
